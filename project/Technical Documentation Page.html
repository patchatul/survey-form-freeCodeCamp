CSS

What is CSS?
  • CSS stands for Cascading Style Sheets.
  • CSS describes how HTML displays on screen or other media.
How do you use CSS with HTML?

There are three ways of inserting a style sheet of CSS into HTML page.

  1. External CSS
    :can be written in any text editor, and must be saved with a .css extension
    Each HTML page must include a reference to the external style sheet file inside the link element, inside the head section.
             h1 {
    color: maroon;
    }
  2. Internal CSS
    : is defined inside the style element, inside the head section..
       style h1 {
    color: maroon;
    }
  3. Inline CSS
    :apply a unique style for a single element.
    h1 style="color:maroon;">
How does CSS syntax look like?
selector { property name:value; }
  • body { background-color: lightblue; }
  • p { color: red; text-align: center; }
  • .header p { font-family: verdana; font-size: 20px; }

don't forget
opening and closing curly braces {} after selector
inside of curly braces, there are property name follows by colon : then value and ends with semicolon;
